When to use release notes vs the rest of the docs

  • Use the main docs pages when you need to learn or run a workflow.
  • Use release notes when you want to know what changed between versions.
  • Use both when you are upgrading and want to understand whether your current workflow or configuration needs attention.

If you are new to Nova, start with Getting Started. If you are already using Nova and want upgrade context, stay here and then jump into the relevant workflow page.
